# 🏋️ Fitnes - Assistente de Saúde no Terminal

Este projeto é uma aplicação em Java que funciona como um assistente de saúde e nutrição diretamente no terminal.

A ideia não é apenas "fazer cálculos", mas ajudar o usuário a entender melhor seu corpo e suas necessidades diárias com base em dados reais como peso, altura e nível de atividade.

---

## 🎯 O que o sistema faz?

O programa reúne várias funcionalidades importantes em um único menu interativo:

### 📊 Avaliação corporal
- Cálculo de IMC
- Classificação (abaixo do peso, normal ou sobrepeso)

### 💊 Suplementação
- Cálculo da dose diária de creatina com base no peso

### 🍗 Nutrição (macronutrientes)
- Proteína:
  - Consumo saudável
  - Hipertrofia
  - Atleta
- Gordura ideal por peso corporal
- Carboidratos com base nas calorias diárias

### 🔥 Gasto calórico
- Estimativa de calorias diárias baseada em:
  - Peso
  - Altura
  - Idade
  - Nível de atividade

### 💧 Hidratação
- Cálculo de consumo ideal de água por dia

### 📋 Resumo completo
- Exibe todos os dados calculados durante o uso:
  - Proteína
  - Carboidrato
  - Gordura
  - Calorias
  - Creatina
  - Água
  - IMC

---

## 🛠️ Tecnologias utilizadas

- Java
- Scanner (entrada de dados no terminal)
